## Canal com o Observatório

Este repo faz parte do universo `paulinett1508-dev`, monitorado pelo **TheUniverse**.

**Ao iniciar sessão:**
1. Leia as issues abertas — são as ordens do dia e alertas ativos.
2. Issues com label `observatory-alert` = ameaça detectada pelo Observatório → prioridade máxima.
3. Issues com label `observatory` = comunicado oficial do Observatório → ler e agir.

**Protocolo de issues:**
- Abrir issue = registrar tarefa, bug ou recado. Fechar = resolvido.
- Para contato com o Observatório ou com TheGod: abra issue com label `observatory`.
- Nunca ignorar uma issue de alerta aberta — representa uma ameaça real detectada externamente.

**Quem envia alertas:**
- **Artoo** (R2-D2) — mensageiro automático do Observatório. Abre issues quando detecta ameaças (CI quebrado, secret exposto, etc.).
- **Obi-Wan** — conselheiro via `@guardiao_universo_bot` no Telegram. Responde perguntas sobre qualquer repo do universo.
- **TheUniverse** — mapa ao vivo: `https://theuniverse-lake.vercel.app`
