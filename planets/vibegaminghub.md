## README (excerpt)

Mini-games para criancas, extraidos do [Super Cartola Manager](https://github.com/paulinett1508-dev/SuperCartolaManagerv5). Deploy via Vercel. · Jogo arcade 8-bit de cobranca de penaltis em canvas. · Canvas interativo fullscreen onde um escorpiao segue o mouse. · Lagarto procedural com inverse kinematics — pernas que caminham de verdade. · Pac-Man
